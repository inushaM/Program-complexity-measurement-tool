Project Resoureces:-
* Netbeans IDE
* visual studio
* XAMPP Server


project running steps:-

1.Run the XAMPP Server
2.execute given sql database called 'spmdb.db' in to the database (open XAMPP server interface and click go to 'Admin' in MySQL module)
3.SPMproject-ws(backend) project open in Netbeans IDE and run using tomcat.
4.React_Frontend(frontend) project open in visual studio.(get the terminal and type 'npm install' after type 'npm start')
